Desmostración del uso de etiquetas de texto en HTML

En esta practica mostraremos el uso de etiquetas de texto que se ocupa en el lenguaje de programación web (HTML), y como se ve en esta pagina.

Como su nombre lo indican, las etiquetas para texto permiten modificar la apariencia que tendrá un bloque de texto al interior de la página. Se puede afectar todo un párrafo, una línea, una palabra, caracteres individuales, simplemente encerrando el respectivo texto dentro de las etiquetas apropiadas. Algunos atributos de texto pueden ser controlados por etiquetas diferentes obteniendo resultados similares.

ETIQUETA <PRE> </PRE>

El lenguaje HTML solamente puede reconocer y mostrar un espacio en blanco entre palabras, independientemente de cuántos se escriban. Tampoco reconoce saltos de línea ni tabuladores, lo que significa que si se introduce el siguiente texto:

Si lo que se desea es que el texto sea visualizado tal como ha sido escrito, debe ser delimitado con la etiqueta <PRE> y </PRE> que indica al navegador que el texto es preformateado
ETIQUETA <BR>

Esta etiqueta permite forzar un salto de línea. Puede observarse que no se está utilizando la etiqueta de cierre </BR>, pues es considerada una etiqueta abierta.

ETIQUETA <P> </P>

En forma equivalente a la etiqueta <BR>, puede utilizarse la etiqueta <P> que puede ser utilizada como etiqueta abierta o cerrada. La apertura indica el inicio de un nuevo párrafo en tanto que </P> indica al navegador que ha terminado dicho párrafo. Si se utiliza como etiqueta abierta, se mostrará una línea en blanco antes de iniciar el párrafo correspondiente. Por el contrario, si se utiliza como etiqueta cerrada, se iniciará un nuevo párrafo, pero sin dejar líneas en blanco previas. Por defecto, un párrafo delimitado con la etiqueta <P> </P> se muestra alineado a la izquierda, situación que puede ser modificada utilizando la etiqueta anidada <ALIGN> según se indica en la siguiente tabla


ETIQUETA <HR>


La etiqueta <HR> (Horizontal Rule) también de tipo abierto, permite dibujar una línea horizontal con tamaño, espesor y alineación modificables. suele utilizarse para separar contenidos.
